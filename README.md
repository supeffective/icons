# @supeffective/icons

Glyph icons SVGs and their React components used in SuperEffective.gg

## Installation

```bash
npm install @supeffective/icons
# or
yarn add @supeffective/icons
# or
pnpm add @supeffective/icons
# or
bun add @supeffective/icons
```
